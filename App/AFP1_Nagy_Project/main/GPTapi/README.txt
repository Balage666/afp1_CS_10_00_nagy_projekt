A Projekt ChatGPT használata:

ELérési lap:
    http://127.0.0.1:8000/gpt/

Felhasználó által beírt üzenet:
    user_input

Számunkra fontos response érték elérése:
    response.choices[0].message.content

Alapértelmezett üzenet a kérés küldése előtt:
    default = ""

    Minden elküldött kérés (a karakterek számától függően) pénzbe kerül, de egy egyszerűbb üzenet kevsebb mint 0.01 USD.
